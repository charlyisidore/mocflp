\subsection{\-Detailed \-Description}
\-Class to represent a {\ttfamily \hyperlink{classFacility}{\-Facility}}. 

\-This class represents a {\ttfamily \hyperlink{classFacility}{\-Facility}} with all its attributes. \-The pair (x,y) representes respectively the first and second coordinates in a bi-\/dimensional geographical space.
